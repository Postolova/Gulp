# Workflow on Gulp

## Build Setup
* ### clone this Repository
##### $ https://github.com/egorov-by-egorov/Gulp.git
* ### Move to Gulp folder
##### $ cd Gulp
* ### install dependencies
##### $ npm install

* ### serve with reload at localhost:8080
##### $ npm start

* ### build for production with minification
##### $ npm run build

* ### Compress images
##### $ npm run compress
